Troubleshooting

🐛 Troubleshooting Guide

🚨 Problem 💡 Solution
📁 File upload fails Check file format (CSV/Excel only)
🕐 Time validation error Ensure "From" time is earlier than "To" time
🔢 Missing columns Follow required format exactly
📄 Document generation fails Check for empty values or wrong data types
🔧 Debug Mode:
streamlit run app.py --logger.level debug
Run this command for detailed error logs and troubleshooting info.
If problems persist, try refreshing the page or contact support for help.
